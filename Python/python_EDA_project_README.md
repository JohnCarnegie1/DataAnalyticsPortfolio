# Exploratory Data Analysis
## Objective
Analyze how penguin species differ from one another, how body mass relates to other physical characteristics, and whether penguins living on different islands show noticeable differences.
## Dataset
penguins dataset from seaborn
## Tools
- Python
- Pandas
- Matplotlib
- Seaborn
- Numpy
## Findings
- Gentoo penguins were generally the largest and heaviest species.
- Body mass showed a positive relationship with flipper length.
- Species distributions varied across islands.
- Physical characteristics differed among species.
